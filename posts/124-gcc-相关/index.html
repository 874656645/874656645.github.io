一、gcc 工作流程#

  1. 预处理 —— -E
    • 宏替换
    • 头文件展开
    • 去掉注释
    • 生成 .i 文件
  2. 编译 —— -S
    • 生成 .s 文件
    • 汇编文件
  3. 汇编 —— -c
    • 生成 .o 文件
    • 二进制文件
  4. 链接

二、gcc 常用参数#

  1. -v / --version
  2. -I : 编译时指定头文件路径
  3. -c : 生成二进制文件 .o
  4. -o : 指定生成的文件名
  5. -g : gdb 调试，生成带有调试信息的二进制文件（Debug模式）
  6. -D : 编译时指定一个宏
  7. -Wall : 显示警告信息
  8. -On : 优化代码，n 是优化级别（1，2，3）
