指针函数和函数指针

December 16, 2021 · 1 min · Rick Cui

  1. 指针函数本质是一个函数，返回值是一个指针；

  2. 函数指针是指向函数的一个指针变量；

#include <iostream>
#include <cstdlib>

using namespace std;

//  指针函数fun
char * fun(const char * p)  {
    cout << "fun: " << p << endl;
    return nullptr;
};

//  函数指针pf 
char * (*pf)(const char * p); 

int main()
{
    pf = fun;                        // 函数指针pf指向函数fun
    pf("Hello"); 
    
    return 0;
}
