Windows 10 中的 Linux 子系统

February 13, 2022 · 1 min · Rick Cui

WSL 的基本命令

  1. 安装 WSL

    wsl --install

  2. 查看可用的 Linu 发行版

    wsl -l -o

  3. 查看已安装的 Linu 分发版

    wsl -l -v

  4. 注销 Linux 发行版

    wsl --unregister Ubuntu

  5. 可以直接从官网下载 Linux 发行版的安装包，然后将 appx 扩展名改为 zip，解压到想要放置的目录，双击 ubuntu.exe
