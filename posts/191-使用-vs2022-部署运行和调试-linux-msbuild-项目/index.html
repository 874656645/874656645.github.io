一、设置 Linux 环境#

1. 确保已安装用于 Visual Studio 的 Linux 开发工作负荷#

2. Linux 所需环境#

sudo apt-get install openssh-server g++ gdb make ninja-build rsync zip

确保 ssh 服务在 Linux 计算机上运行

sudo service ssh start

参考官方文档

二、连接到远程 Linux#

打开【工具】>【选项】>【跨平台】>【连接管理器】进行添加设置

也可以连接到 WSL，参考官方文档
