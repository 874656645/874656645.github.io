Linux 目录介绍

February 10, 2022 · 1 min · Rick Cui

bin 二进制文件，可执行文件
boot 系统开机启动项
dev device 设备（硬盘、显卡、显示器 一切皆文件）
lib 动态库
mnt 手动挂载目录
media 外设自动挂载目录
root 超级用户的家目录
usr unix system resource 头文件include、源文件src、用户安装程序/usr/local
etc 配置文件
opt 安装第三方应用程序
home Linux 所有用户的家目录（操作系统的家目录）
tmp 临时文件目录，系统重启后会被清空
