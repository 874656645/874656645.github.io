Ubuntu 编译指定版本的 curl 库

November 3, 2023 · 1 min · Rick Cui

  1. 卸载原有版本的 curl

    apt-get --purge remove curl

  2. 下载所需版本的 curl

    wget https://curl.se/download/curl-7.47.1.tar.gz

  3. 解压并编译

    tar -zxvf curl-7.47.1.tar.gz
cd curl-7.47.1
./configure shared
make
make install

  4. 编译好的可执行文件在 /usr/local/bin 目录下

    编译好的动态库和静态库在 /usr/local/lib 目录下
