在python中使用virtualenv创建虚拟环境

注意：要使用cmd命令行，不能使用Power Shell

1、使用pip安装

pip install virtualenv

2、创建项目目录并用cd进入

mkdir E:\python\Virtualenv
cd E:\python\Virtualenv

3、创建虚拟环境

virtualenv -p c:\Python36\python.exe env
或者
virtualenv env
此命令会在当前目录下创建一个env目录

4、启动虚拟环境

env\Scripts\activate

5、离开虚拟环境

deactivate

6、删除env文件夹即可
